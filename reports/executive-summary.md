# Executive Summary

This release packages a compliance-first index kit for W3Schools-related research deliverables.

## Key points

- This kit does NOT include W3Schools tutorial text or copied content.
- The included index data is representative sample metadata only (URLs + classification).
- Automated crawling/scraping of W3Schools is prohibited by their Terms; any fetch tooling is disabled by default.
- The site is deployed at forge.ameti.one via Vercel.

## What's included

- URL-level metadata index for W3Schools public pages
- Content taxonomy classifying tutorials, references, exercises, quizzes, and tools
- Compliance documentation and takedown procedures
- Reproducible build pipeline and deployment configurations
