# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

**Email:** security@ameti.one

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response timeline

- **Acknowledgment:** within 2 business days
- **Assessment:** within 5 business days
- **Resolution:** based on severity, typically within 14 days for critical issues

## Scope

This policy covers:
- The static site hosted at forge.ameti.one
- Build scripts and CI/CD pipelines in this repository
- Data validation and compliance tooling

## Out of scope

- Third-party services (Vercel, GitHub, etc.)
- W3Schools infrastructure (we have no control over it)
